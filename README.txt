The CIA triad describes three aspects of security that are important to a company if they value their data.
The C is for Confidentiality and is for keeping data protected as well as keeping it away from those who do not need to see it.
The I is for Integrity and means keeping the data consistent and accurate with current real world knowledge so that it stays reliable.
The A is for Availability and is meant to explain how data should be kept as available as possible to those who may need to access it.
